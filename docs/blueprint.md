# **App Name**: RefillWise

## Core Features:

- Frontend Hosting: Host the Next.js frontend application code from https://github.com/idealchop/smartrefill-webapp via the /app-route.
- BFF Functionality: Implement backend-for-frontend (BFF) logic using reusable functions from https://github.com/idealchop/river-bff.
- Microservice API: Utilize microservice APIs from https://github.com/idealchop/river-microservice for backend services and functionalities.

## Style Guidelines:

- Primary color: HSL(210, 70%, 50%) translating to a vibrant blue (#3399FF) to communicate trust and efficiency in the refill process.
- Background color: HSL(210, 20%, 95%) translating to a very light blue (#F0F8FF) to maintain a clean, airy feel.
- Accent color: HSL(180, 60%, 40%) translating to a bright cyan (#40BFBF) used to highlight important interactive elements like buttons.
- Body font: 'Inter', a sans-serif font known for its clean and modern appearance.
- Headline font: 'Space Grotesk', a sans-serif font for headlines to create a modern, precise style
- Use a set of consistent, outline-style icons throughout the app to represent refill types, locations, and settings.
- Employ a clean, card-based layout for displaying refill options and history to improve readability and interaction.
- Use subtle transitions and loading animations to enhance user experience.
